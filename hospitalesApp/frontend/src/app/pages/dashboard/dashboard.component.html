Hospitales APP

Back-end armado en Node-JS y MongoDB usando distintintos tipos de librerías, algunas de estas son:

  • Moongose: Administrador de BD
  • Json Web Token: Seguridad de autenticación via Token
  • Google Api: Autenticacíon via google consumiendo su API

Front-End hecho completamente en Angular utilizando los principales patrones de este Framework:

  • Lazyload: carga de componentes según necesidad. Alivianando así la fluidez de la app
  • Observables, Manejo del estado y comunicación entre componentes mediante Inputs y Outputs
  • Interceptores y Guards

Principales funciones de la app:

  • ABM Completo de hospitales y médicos (solo para usuarios con rol ADMIN)
  • Modificación de Roles (solo para usuarios rol ADMIN)
  • Validación para no poder borrarse a uno mismo
  • Servicio aparte para carga de imagenes y modificaciones de las mismas (Usuarios, Hospitales y Medicos)
  • Autenticación via JWT y generación de nuevo token cuando está por expirar
  • Autenticación via Google
  • Bloqueo de rutas y acciones según ROL de usuario
  • Creación de nuevos usuarios
  • Ajustes de app: Cambiar el color total del tema de la app
